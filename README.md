# danyeong-lee.github.io

Personal blog — AI research notes (drug discovery, protein models, LLM interpretability).

Built with Jekyll + minima, hosted on GitHub Pages.

## Local preview

```bash
bundle install
bundle exec jekyll serve --drafts
```

## Layout

- `_posts/` — published posts (`YYYY-MM-DD-slug.md`)
- `_drafts/` — unpublished drafts; template in `_drafts/2026-02-19-post-template.md`
- `assets/images/` — post images
- `about.md` — about page
- `docs/blog_workflow.md` — authoring workflow
